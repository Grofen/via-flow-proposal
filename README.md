# ViaFlow Proposal

This repository contains the ViaFlow software development proposal for VIA Fahrdienst GmbH.

## GitHub Pages Setup

To enable GitHub Pages for this repository:

1. Go to the repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click "Save"

The proposal will be available at: `https://grofen.github.io/via-flow-proposal/`

## Local Viewing

Simply open `index.html` in your web browser to view the proposal locally.
